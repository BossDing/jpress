# About
基于JPress V0.5.0版本定制，建立这个非maven版本主要是为了两点：<br>
*一是为了方便新手能够快速运行，<br>
*二是方便主题制作的时候能够热部署。<br>
<br>
删除了源码中的war包，需要下载安装包的，可以去JPress之家下载最新的安装包。
